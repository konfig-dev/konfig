# Welcome!

This is an interactive demo for Konfig.

### How to use this demo

On the left, read the directions and follow the steps.

On the right, you'll find a development environment to execute commands and edit
files.

At the top of the development environment, you will see a file editor.

![Editor](/img/editor.png)

And at the bottom of the development environment, you will see a terminal.

![Terminal](/img/terminal.png)

Each step will present an exercise designed to teach you how to generate SDKs
with Konfig. Later steps require the completion of earlier ones, so you must go
from start to finish.

Click the button below to go to the next step.
