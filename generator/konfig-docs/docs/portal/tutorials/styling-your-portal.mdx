# Syling your Portal

You can configure:

<CH.Section>

1. [The title of the portal](focus://2)
2. [The primary color of the portal](focus://3)
3. [The favicon of the portal](focus://4)
4. [The logo of the portal](focus://5)
5. [The light/dark mode logo of the portal (preferred)](focus://6:8)

<Admonition type="note">
If providing a light and dark logo, ensure the light logo has dark text while
the dark logo has have light text.
</Admonition>

```yaml konfig.yaml
portal:
  title: Acme
  primaryColor: '#9fc1be'
  favicon: favicon.png
  logo: portal-logo.png
  # logo:
  #   light: portal-logo-light.png
  #   dark: portal-logo-dark.png
```
</CH.Section>
